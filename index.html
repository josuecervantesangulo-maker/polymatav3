🌐 GitHub Pages — App con URL propia
Con GitHub Pages tu app vive en una URL fija tipo tuusuario.github.io/polymata y el localStorage persiste siempre, en cualquier dispositivo que use esa URL.
1
Ve a github.com y crea una cuenta gratis
2
Crea un repositorio nuevo llamado polymata
3
Sube este archivo polymata_v4.html renombrado como index.html
4
Ve a Settings → Pages → Source: main branch → Save
✅
Tu app estará en tuusuario.github.io/polymata — instálala como PWA desde ahí
⚠️ Por qué se borran los datos: Cuando abres el HTML como archivo local (file://), Chrome en Android no garantiza persistir el localStorage. La solución es usar una URL real (GitHub Pages) o hacer backup frecuente.
